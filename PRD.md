## 1. 제품 개요

### 1.1 제품명
**Reel Maker AI** (가칭)

### 1.2 제품 비전
누구나 쉽게 트렌디한 인스타그램 릴스를 자동으로 생성할 수 있는 AI 기반 콘텐츠 제작 플랫폼

### 1.3 제품 설명
키워드만 입력하면 AI가 자동으로 대본을 작성하고, 관련 이미지/영상을 수집하여 음성, 자막, 효과가 포함된 완성도 높은 인스타그램 릴스를 생성해주는 서비스입니다.

### 1.4 핵심 가치 제안
- ⏱️ **시간 절약**: 수시간 걸리는 영상 제작을 5분으로 단축
- 🎨 **전문성**: AI가 트렌드를 반영한 전문적인 콘텐츠 생성
- 💰 **비용 절감**: 비싼 영상 제작 툴이나 디자이너 불필요
- 📈 **트렌드 반영**: 최신 키워드와 트렌드 자동 반영

## 2. 문제 정의

### 2.1 현재 문제점
1. **시간 소요**: 릴스 제작에 평균 2-4시간 소요
2. **전문 지식 필요**: 영상 편집 스킬이 없으면 퀄리티 낮음
3. **아이디어 고갈**: 지속적인 콘텐츠 아이디어 발굴 어려움
4. **리소스 부족**: 적절한 이미지/영상 소스 찾기 어려움
5. **일관성 부족**: 콘텐츠 톤앤매너 유지 어려움

### 2.2 타겟 페인 포인트
- "매일 릴스를 올리고 싶은데 시간이 없어요"
- "영상 편집을 못해서 퀄리티가 떨어져요"
- "어떤 내용으로 만들지 아이디어가 없어요"
- "저작권 걱정 없는 이미지를 찾기 어려워요"

## 3. 목표 및 성공 지표

### 3.1 비즈니스 목표
- **Phase 1 (3개월)**: MVP 출시 및 1,000명의 베타 사용자 확보
- **Phase 2 (6개월)**: 월 활성 사용자 10,000명 달성
- **Phase 3 (12개월)**: 유료 전환율 5% 달성

### 3.2 핵심 성공 지표 (KPI)

#### 사용자 지표
- **MAU (월 활성 사용자)**: 10,000명 (6개월 내)
- **DAU/MAU 비율**: 30% 이상
- **사용자 유지율**: 30일 유지율 40% 이상

#### 제품 지표
- **영상 생성 성공률**: 95% 이상
- **평균 생성 시간**: 3분 이내
- **사용자 만족도**: NPS 50 이상

#### 비즈니스 지표
- **유료 전환율**: 5% (12개월 내)
- **월 매출**: $10,000 (12개월 내)
- **고객 생애 가치 (LTV)**: $50 이상

## 4. 타겟 사용자

### 4.1 주요 페르소나

#### 페르소나 1: 개인 크리에이터 민지
- **나이**: 25세
- **직업**: 프리랜서 인플루언서
- **목표**: 일일 1-2개의 릴스를 효율적으로 제작
- **페인 포인트**: 시간 부족, 아이디어 고갈
- **기술 수준**: 중급 (영상 편집 기초 가능)

#### 페르소나 2: 마케터 준호
- **나이**: 32세
- **직업**: 스타트업 마케팅 매니저
- **목표**: 브랜드 홍보용 릴스 대량 제작
- **페인 포인트**: 예산 제약, 외주 비용 부담
- **기술 수준**: 초급 (영상 편집 경험 없음)

#### 페르소나 3: 소상공인 수영
- **나이**: 38세
- **직업**: 카페 운영
- **목표**: 가게 홍보용 릴스 주 2-3회 게시
- **페인 포인트**: 시간 없음, 전문 지식 없음
- **기술 수준**: 초급

### 4.2 사용자 세그먼트

| 세그먼트 | 비중 | 특징 | 니즈 |
|---------|------|------|------|
| 개인 크리에이터 | 40% | 높은 활동성, 품질 중시 | 빠른 제작, 다양한 스타일 |
| 마케터/에이전시 | 35% | 대량 제작, ROI 중시 | 배치 처리, 브랜드 일관성 |
| 소상공인/자영업 | 25% | 낮은 기술력, 가성비 중시 | 쉬운 사용, 저렴한 가격 |

## 5. 기능 요구사항

### 5.1 핵심 기능 (MVP - Phase 1)

#### F1. 키워드 기반 콘텐츠 생성
- **설명**: 사용자가 입력한 키워드를 기반으로 릴스 대본 자동 생성
- **우선순위**: P0 (필수)
- **상세 요구사항**:
  - 키워드 입력 (최소 1개, 최대 5개)
  - GPT-4 API를 활용한 대본 생성
  - 대본 구조: 훅(3초) → 본문(20초) → 클로징(7초)
  - 생성 시간: 10초 이내
  - 사용자 대본 수정 가능

**인수 기준**:
- [ ] 키워드 입력 시 5초 내 대본 생성
- [ ] 생성된 대본이 30초 분량의 적절한 길이
- [ ] 대본 수정 기능 동작
- [ ] 부적절한 키워드 필터링

#### F2. 이미지/영상 자동 수집
- **설명**: 대본 내용에 맞는 이미지 및 영상 클립 자동 검색 및 다운로드
- **우선순위**: P0 (필수)
- **상세 요구사항**:
  - Unsplash, Pexels API 연동
  - 장면당 3-5장의 이미지 수집
  - 저작권 확인 (상업적 이용 가능한 것만)
  - 이미지 품질 필터링 (최소 1920x1080)
  - 사용자 이미지 업로드 가능 (선택)

**인수 기준**:
- [ ] 장면당 적절한 이미지 자동 선택
- [ ] 모든 이미지가 상업적 이용 가능
- [ ] 이미지 다운로드 성공률 95% 이상
- [ ] 수동 이미지 교체 가능

#### F3. TTS (음성 생성)
- **설명**: 대본을 자연스러운 음성으로 변환
- **우선순위**: P0 (필수)
- **상세 요구사항**:
  - ElevenLabs 또는 Google Cloud TTS 연동
  - 한국어 음성 지원 (남성/여성 선택)
  - 자연스러운 억양 및 속도
  - 음성 프리뷰 기능
  - 음성 변경 가능

**인수 기준**:
- [ ] 대본을 명확한 음성으로 변환
- [ ] 음성 생성 시간 20초 이내
- [ ] 남성/여성 음성 선택 가능
- [ ] 음성 품질 만족도 4.0/5.0 이상

#### F4. 자막 생성 및 오버레이
- **설명**: 음성에 맞춰 자막 자동 생성 및 영상에 합성
- **우선순위**: P0 (필수)
- **상세 요구사항**:
  - 단어 단위 자막 타이밍 동기화
  - 3가지 자막 스타일 제공
  - 폰트, 색상, 위치 조정 가능
  - 자막 애니메이션 효과

**인수 기준**:
- [ ] 음성과 자막 완벽 동기화
- [ ] 가독성 좋은 자막 디자인
- [ ] 자막 스타일 변경 가능

#### F5. 영상 합성 및 렌더링
- **설명**: 이미지, 음성, 자막을 합쳐 최종 릴스 영상 생성
- **우선순위**: P0 (필수)
- **상세 요구사항**:
  - 출력 포맷: MP4 (H.264)
  - 해상도: 1080x1920 (9:16)
  - 프레임레이트: 30fps
  - 전환 효과 (페이드, 슬라이드 등)
  - Ken Burns 효과 (줌/패닝)
  - 배경음악 자동 선택 및 삽입
  - 렌더링 시간 표시

**인수 기준**:
- [ ] 30초 영상 3분 내 렌더링
- [ ] 릴스 규격에 맞는 출력
- [ ] 영상 품질 저하 없음
- [ ] 렌더링 성공률 98% 이상

#### F6. 미리보기 및 다운로드
- **설명**: 생성된 영상 미리보기 및 다운로드
- **우선순위**: P0 (필수)
- **상세 요구사항**:
  - 웹 브라우저에서 미리보기
  - MP4 파일 다운로드
  - 썸네일 이미지 자동 생성
  - 추천 해시태그 제공 (10-15개)

**인수 기준**:
- [ ] 미리보기 재생 원활
- [ ] 다운로드 성공률 100%
- [ ] 해시태그 관련성 높음

### 5.2 부가 기능 (Phase 2)

#### F7. 배치 처리
- **설명**: 여러 키워드를 한 번에 입력하여 다수의 릴스 동시 생성
- **우선순위**: P1
- **상세 요구사항**:
  - 최대 10개 키워드 동시 처리
  - 진행 상황 표시
  - 완료 알림

#### F8. 스타일 프리셋
- **설명**: 사전 정의된 릴스 스타일 템플릿 제공
- **우선순위**: P1
- **상세 요구사항**:
  - 5가지 스타일 (뉴스형, 교육형, 엔터테인먼트형, 미니멀, 다이나믹)
  - 커스텀 스타일 저장

#### F9. 배경음악 라이브러리
- **설명**: 저작권 없는 배경음악 선택 기능
- **우선순위**: P1
- **상세 요구사항**:
  - 50개 이상의 무료 음원
  - 장르별 분류
  - 음악 프리뷰

#### F10. 브랜드 키트
- **설명**: 기업/개인 브랜드 요소 저장 및 일괄 적용
- **우선순위**: P1
- **상세 요구사항**:
  - 로고 삽입
  - 브랜드 컬러 적용
  - 인트로/아웃트로 설정

### 5.3 고급 기능 (Phase 3)

#### F11. AI 이미지 생성
- **설명**: DALL-E 또는 Midjourney를 활용한 커스텀 이미지 생성

#### F12. 인스타그램 자동 업로드

#### F13. 성과 분석 대시보드

- **우선순위**: P3
- **예상 기간**: Phase 3

## 6. 비기능 요구사항

### 6.1 성능 요구사항

| 항목 | 요구사항 | 측정 방법 |
|------|---------|-----------|
| 대본 생성 시간 | 10초 이내 | API 응답 시간 |
| 영상 렌더링 시간 | 30초당 3분 이내 | 렌더링 완료 시간 |
| 페이지 로딩 시간 | 2초 이내 | Lighthouse 점수 |
| 동시 사용자 처리 | 100명 | 부하 테스트 |
| API 응답 시간 | 평균 500ms 이하 | 모니터링 |

### 6.2 확장성 요구사항
- **수평 확장**: 사용자 증가에 따라 서버 증설 가능
- **큐 시스템**: 렌더링 작업 큐잉 (Celery, Redis)
- **CDN 활용**: 정적 파일 및 생성된 영상 배포

### 6.3 보안 요구사항
- **인증**: JWT 기반 사용자 인증
- **데이터 암호화**: HTTPS 통신, 민감 정보 암호화
- **API 키 보호**: 환경 변수 관리
- **입력 검증**: SQL Injection, XSS 방지
- **Rate Limiting**: API 남용 방지 (사용자당 시간당 10개)

### 6.4 사용성 요구사항
- **직관적 UI**: 3번의 클릭으로 영상 생성 완료
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원
- **다국어 지원**: 한국어, 영어 (Phase 2)
- **접근성**: WCAG 2.1 AA 준수

### 6.5 신뢰성 요구사항
- **가용성**: 99.5% 이상 (월 3.6시간 다운타임 허용)
- **백업**: 일일 자동 백업
- **에러 처리**: 명확한 에러 메시지 및 복구 가이드
- **모니터링**: Sentry, CloudWatch 활용

### 6.6 호환성 요구사항
- **브라우저**: Chrome, Safari, Firefox, Edge (최신 2개 버전)
- **OS**: Windows, macOS, iOS, Android
- **Instagram API**: Graph API v18.0 이상

## 7. 사용자 시나리오

### 7.1 시나리오 1: 첫 릴스 생성 (신규 사용자)

**배경**: 민지는 인플루언서로 활동 중이며, AI 트렌드에 대한 릴스를 만들고 싶어합니다.

**단계**:
1. 웹사이트 접속 및 회원가입
2. "AI 트렌드 2025" 키워드 입력
3. 영상 길이 선택 (30초)
4. 음성 선택 (여성, 밝은 톤)
5. "생성하기" 버튼 클릭
6. 3분 대기 (진행 상황 표시)
7. 미리보기로 결과 확인
8. 만족스러워서 다운로드
9. 해시태그 복사
10. 인스타그램에 수동 업로드

**예상 소요 시간**: 5분

**결과**: 민지는 별다른 노력 없이 전문적인 릴스를 만들었습니다.

### 7.2 시나리오 2: 커스터마이징 (중급 사용자)

**배경**: 준호는 회사 마케팅 담당자로 브랜드 톤에 맞는 릴스를 만들어야 합니다.

**단계**:
1. 로그인
2. "친환경 패션" 키워드 입력
3. 자동 생성된 대본 확인 및 일부 수정
4. 이미지 중 2개를 회사 제공 이미지로 교체
5. 자막 스타일 변경 (브랜드 컬러)
6. 배경음악 변경 (잔잔한 음악)
7. 생성 및 미리보기
8. 다운로드
9. 브랜드 키트로 설정 저장 (다음 사용을 위해)

**예상 소요 시간**: 8분

**결과**: 준호는 브랜드 아이덴티티가 반영된 맞춤형 릴스를 제작했습니다.

### 7.3 시나리오 3: 배치 처리 (고급 사용자)

**배경**: 수영은 카페를 운영하며 주간 프로모션용 릴스 5개를 한 번에 만들고 싶어합니다.

**단계**:
1. 로그인
2. 배치 모드 선택
3. 5개 키워드 입력
   - "월요일 커피 특가"
   - "신메뉴 아이스 라떼"
   - "원두 이야기"
   - "카페 인테리어"
   - "주말 브런치"
4. 공통 설정 (음성, 길이, 스타일)
5. 일괄 생성 시작
6. 15분 대기 (5개 영상)
7. 각 영상 미리보기 및 다운로드
8. 해시태그 일괄 복사

**예상 소요 시간**: 20분

**결과**: 수영은 한 주치 콘텐츠를 한 번에 효율적으로 준비했습니다.

## 8. UI/UX 요구사항

### 8.1 정보 아키텍처

```
홈페이지
├── 헤더
│   ├── 로고
│   ├── 주요 기능 (릴스 만들기, 내 작품, 요금제)
│   └── 로그인/회원가입
├── 히어로 섹션
│   ├── 메인 카피
│   ├── CTA 버튼 (무료로 시작하기)
│   └── 데모 영상
├── 기능 소개
├── 사용 방법 (3단계)
├── 요금제
└── 푸터

릴스 생성 페이지
├── Step 1: 키워드 입력
├── Step 2: 옵션 선택
├── Step 3: 생성 및 대기
└── Step 4: 결과 및 다운로드
```

### 8.2 화면별 상세 요구사항

#### 8.2.1 홈페이지
- **히어로 섹션**
  - 메인 카피: "5분 만에 바이럴 릴스 완성"
  - 서브 카피: "AI가 대본부터 편집까지 모두 해드립니다"
  - CTA 버튼: 명확하고 눈에 띄게 ("무료로 시작하기")
  - 데모 영상: 10초짜리 제품 사용 영상

#### 8.2.2 릴스 생성 페이지
- **Step 1: 키워드 입력**
  - 큰 입력창
  - 플레이스홀더: "예: AI 트렌드, 건강한 다이어트, 여행 팁"
  - 추천 키워드 버튼 (트렌딩 키워드)
  - 진행 단계 표시 (1/4)

- **Step 2: 옵션 선택**
  - 영상 길이: 15초 / 30초 / 60초
  - 음성 선택: 남성 / 여성 (각 3가지 톤)
  - 스타일: 5가지 프리셋 (아이콘과 설명)
  - 고급 설정 (접기/펼치기)

- **Step 3: 생성 및 대기**
  - 진행률 표시 (%)
  - 현재 작업 표시 ("대본 생성 중...", "이미지 수집 중...")
  - 예상 대기 시간
  - 취소 버튼

- **Step 4: 결과 및 다운로드**
  - 큰 영상 플레이어 (중앙)
  - 재생/일시정지 컨트롤
  - 다운로드 버튼 (큼직하게)
  - 수정하기 버튼
  - 해시태그 박스 (복사 버튼)
  - 공유 버튼

### 8.3 디자인 가이드라인

#### 색상 팔레트
- **Primary**: #6366F1 (인디고)
- **Secondary**: #EC4899 (핑크)
- **Success**: #10B981 (그린)
- **Warning**: #F59E0B (앰버)

- **Neutral**: #6B7280, #F3F4F6 (그레이 스케일)

- **입력창**: 테두리, 포커스 시 색상 변화

## 9. 제약사항 및 가정

### 9.1 제약사항

#### 기술적 제약
- Instagram Graph API는 개인 계정을 지원하지 않음 (비즈니스 계정만)
- TTS API 일일 사용량 제한 (ElevenLabs: 10,000 chars/day)
- 이미지 API Rate Limit (Unsplash: 50 req/hour)
- 영상 렌더링은 서버 리소스 집약적

#### 법적 제약
- 저작권 있는 음악/이미지 사용 불가
- GDPR 및 개인정보보호법 준수 필요
- 유료 API 이용 약관 준수

#### 비즈니스 제약
- MVP 개발 예산: $10,000
- 출시 기한: 3개월
- 개발 인력: 2명 (풀스택 1, 디자이너 1)

### 9.2 가정

#### 사용자 행동
- 사용자는 인스타그램 계정을 보유하고 있음
- 사용자는 기본적인 웹 사용 능력이 있음
- 사용자는 키워드를 명확히 입력할 수 있음

#### 기술 환경
- 사용자는 안정적인 인터넷 연결을 보유
- 최신 브라우저 사용 (Chrome, Safari)
- 평균 다운로드 속도 10Mbps 이상

#### 시장 환경
- 인스타그램 릴스 트렌드는 지속될 것
- AI 기술 발전이 계속될 것
- 경쟁 제품이 3-6개월 내 출시되지 않을 것
